```
## Folder Usage
/lib: a folder that provides all the fonts and theme for the app.
/public (outside): a folder that contains all the static files for the app.
/components: the main component file for the app, containing global components
/route/components: folders within different routing that contains the components for the route

## Files Usage
page.tsx: the main file for the route or app
/components/page.tsx: the main component file for the route
/components/layout.tsx: the layout component file for the route
/components/test.ts: the test file for the component
```
